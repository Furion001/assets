1. Installing Master Sword
Installing Master Sword is very simple. When starting up the installation program, you will be asked
where you want the game to be installed. The default folder is C:\Sierra\Half-Life.

After the installation, you will have a folder called \ms in your Half-Life directory.
A shortcut should normally be created on your desktop saying "Master Sword".
If not, start Half-Life normally, click Custom Games and Activate Master Sword.

2. Controls
After you have installed the game you can set the controls to your liking.
If you want to change them ingame, type "bind X command", where X
is your desired key, and the command one of the console commands below:
